# ModelSearchCore

ModelSearchCore es una plataforma modular para la gestión, orquestación y despliegue de flujos de trabajo de machine learning. El proyecto está compuesto por servicios independientes para entrenamiento, orquestación de jobs y gestión de experimentos, integrando tecnologías modernas como MLflow, Kubernetes y notificaciones automatizadas.

Incluye:
- Worker para procesamiento de jobs de entrenamiento.
- Orquestador de jobs con soporte reactivo.
- Servicio dedicado a MLflow para el manejo de experimentos.
- Integración con AWS, Kubernetes y notificaciones vía Discord.

Este repositorio está diseñado para facilitar la escalabilidad, trazabilidad y automatización en proyectos de ciencia de datos y machine learning.
